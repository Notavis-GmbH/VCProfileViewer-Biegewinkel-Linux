# VC3DProfileViewer – Biegewinkel (Linux ARM64 / Raspberry Pi 5)

Notavis GmbH — automatischer Build für Raspberry Pi 5 (aarch64).

## Installation

```bash
curl -sL https://raw.githubusercontent.com/Notavis-GmbH/VCProfileViewer-Biegewinkel-Linux/main/install.sh | bash
```

Gleicher Befehl für Updates — der Installer erkennt automatisch ob eine neue Version vorliegt.

## Manuell

```bash
tar -xzf VCProfileViewer_Biegewinkel_linux_arm64_v1.0-*.tar.gz
./start.sh
```
